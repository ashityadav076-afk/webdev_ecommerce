# AY Store - E-Commerce Website

A modern and responsive e-commerce website developed using HTML, CSS, and JavaScript. The project demonstrates front-end web development concepts including dynamic product rendering, shopping cart management, wishlist functionality, category filtering, and responsive user interface design.

## Project Overview

AY Store is a fully functional front-end e-commerce platform that provides users with an engaging online shopping experience. The website includes product browsing, category-based filtering, shopping cart operations, wishlist management, promotional banners, customer reviews, and newsletter subscription features.

Developed as a Web Development Project under SOCE, IIT Kanpur.
